## 📌 About This Repository

This repository contains all the **practical lab sheets and SQL implementations** for the **Database Management Systems (DBMS) Lab** course.

The lab focuses on important database concepts including:

- ER Diagram Design
- Relational Schema Design
- DDL (Data Definition Language)
- DML (Data Manipulation Language)
- TCL (Transaction Control Language)
- DCL (Data Control Language)
- JOIN Operations
- GROUP BY and ORDER BY
- SQL Transactions (COMMIT, ROLLBACK, SAVEPOINT)
